CKEditor Super
===============

Features:
- Type any CSS class on the selected element.
- Dropdown of admin-defined style suggestions.
- Insert predefined HTML templates / blocks.
- Admin UI to manage suggestions and templates.
- Optionally load frontend editor CSS for WYSIWYG appearance.

Install:
1. Extract to web/modules/custom/ckeditor_super
2. Enable module: drush en ckeditor_super -y
3. Configure at /admin/config/content/ckeditor-super
